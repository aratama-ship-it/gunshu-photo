# 集合写真アシスタント（並び方プレビュー）

集合写真を撮るときの **推奨F値・並び方・被写界深度** を、その場で試算するためのプロトタイプです。
人数・被写体距離・焦点距離・センサーサイズ（フルサイズ / APS-C）を動かすと、以下がリアルタイムに更新されます。

- **推奨絞り**（安全マージン込み）と、物理計算そのままの**最低F値**
- **列数・各列の人数・並び**（前列→後列）
- **画角内の並びプレビュー**（横位置 3:2、身長165cm・肩を寄せた密着前提、後列ほど少し暗く）
- **ピントが合う範囲**（被写界深度の前後端）と集団の奥行き

## 使い方

`index.html` をブラウザで開くだけです（ビルド不要の静的サイト）。

## 前提・注意

- 被写界深度は許容錯乱円 0.03mm（フルサイズ）/ 0.02mm（APS-C）＝一般的な鑑賞サイズ基準で計算しています。
  高画素機の等倍鑑賞ではより厳しくなるため、推奨値には安全マージン（+1段）を上乗せしています。
- 「安全側の目安」を出すツールです。会場の形状・身長差・主役配置などは現場で調整してください。

## 構成

| ファイル | 役割 |
|---|---|
| `index.html` | 画面構造 |
| `style.css` | ダーク・プロカメラ風のスタイル |
| `script.js` | 計算ロジック・プレビュー描画（調整パラメータは冒頭の `CONFIG` に集約） |
